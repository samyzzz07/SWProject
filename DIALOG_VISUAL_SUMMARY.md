## Color Scheme

All dialogs use consistent colors:
- **Primary Blue (#2196F3)**: View Teams, Check Schedule, Export buttons
- **Green (#4CAF50)**: Approve Team, Approve actions, Success indicators
- **Orange (#FF9800)**: Publish Schedule
- **Purple (#9C27B0)**: Collect Fees, Fee-related actions
- **Cyan (#00BCD4)**: Find Sponsor
- **Teal (#009688)**: Check Schedule validation
- **Indigo (#3F51B5)**: Assign Referees
- **Red (#f44336)**: Postpone Match, Reject actions, Warnings
- **Gray (#607D8B)**: Request Menu, Secondary actions

## Layout Features

All dialogs share these common features:
- **Title bar**: Clear identification of dialog purpose
- **Separator lines**: Visual separation of sections
- **Consistent spacing**: 15px between sections, 10px within sections
- **Padding**: 20px around dialog content
- **Button alignment**: Right-aligned at bottom
- **Responsive tables**: Auto-adjust column widths
- **Form layouts**: GridPane for label-value pairs
- **Input validation**: Client-side validation before submission
